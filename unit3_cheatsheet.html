Select Statements

Select statments are used in SQL to grab specfic information from a database/table.

SELECT column_name FROM table_name
WHERE column_name = 'username'

Terms

WHERE

Operators:

  • =equal
  • !=not equal
  • >greater than
  • <less than
  • >greater than or equal to
  • <less than or equal to
  • BETWEENbetween (range is inclusive)
  • LIKEsearch for a pattern
  • INspecifies multiple possible values for a column
SELECT column_name, function(column)
FROM table_name
WHERE column_name operator value
Distinct
Lists distinct values of the column (no duplicates)
SELECT DISTINCT column_name FROM table_name
Group By
used to group what results you pulled by one or more columns
SELECT column_name, function(column)
FROM table_name
WHERE column_name operator value
GROUP BY column_name
Count
returns number of values in the column
SELECT COUNT(column_name)
FROM table_name
Sum
returns total of a column (has to be numbers)
SELECT SUM(column_name)
FROM table_name
Having
like 'WHERE' but you can use functions
SELECT column_name, function(column)
FROM table_name
WHERE column_name operator value
GROUP BY column_name
HAVING function(column) operator value
Join
used to combine rows from multiple tables, based on a common field
SELECT column_name(s)
FROM table1
JOIN table2
ON table1.column_name = table2.column_name
