Ruby: Enumerable#cycle

Date: April 4, 2014

Cycle is a method in ruby that calls a block to each element of the enumberable, a set or infinite number of times. It doesn't do anything if the number of times isn't positive or the element is empty. If the loop finishes sucessfully, then it returns nil.
The syntax looks like this:

.cycle(n){|obj| block }

An example of how it is used is as follows

myarray = ["one", "two", "three"]

myarray.cycle {|x| puts x }
# prints one, two, three, one, two, three.. forever.

myarray.cycle(2) {|x| puts x }
# prints one, two, three, one, two, three.

An instance of when you would use this is when you want to cycle through an array a set number of times and do something to the input. For example if you wanted to cycle through an array of names, capitalize them and print them.

myarray = ["amy", "betty", "cathy"]

myarray.cycle(1) {|x| puts x.capitalize! }
# prints "Amy", "Betty", "Cathy" once and returns nil
